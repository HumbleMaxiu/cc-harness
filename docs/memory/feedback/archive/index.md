# Feedback Archive Index

> 存放已完成汇总的 feedback 月度归档。长期 memory 只保留最近待处理项、近期已处理项和当前有效规则；更早的记录应迁移到这里。

## 使用规则

1. `agent-feedback.md` 保留最近待处理项与近期已处理项，不充当永久流水账。
2. 当某月累计记录较多，或当前文件开始影响会话恢复效率时，将已完成项汇总进当月归档。
3. 月度归档保留：
   - 该月高频问题模式
   - 已执行的代表性修复类型
   - 升级为 `prevents-recurrence` 的条目
   - 仍未关闭的跨任务风险
4. 原始证据仍以交接文档和命令输出为准；archive 只保留摘要和趋势。
