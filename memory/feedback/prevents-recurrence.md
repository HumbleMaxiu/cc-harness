# 防止再犯记录

> 记录频繁出现的问题及其预防措施，防止同类错误重复发生。

## 规范

### 记录格式

每条记录包含：
- **问题类型** (problem type)
- **问题描述** (description)
- **预防措施** (prevention — 写入 AGENTS.md 或相关规范)
- **首次出现** (first occurrence date)
- **出现次数** (occurrence count)

### 更新时机

当同一类型的问题出现 **2次或以上** 时，将预防措施写入：
1. `AGENTS.md` 的行为规则
2. 相关 Agent 的定义文件
3. 或创建专门的规范文档
