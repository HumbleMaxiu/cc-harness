# Agent 反馈

## 处理原则

> Agent 反馈需要询问用户后才能执行。

## 与用户反馈的区别

| | 用户反馈 | Agent 反馈 |
|--|---------|-----------|
| 优先级 | 最高 | 次高 |
| 执行 | 无需确认 | 需询问用户 |
| 来源 | 用户 | Agent 自检 / Reviewer / Tester |

- 完整记录：`memory/feedback/agent-feedback.md`
- 处理指南：`docs/feedback/feedback-collection.md`
