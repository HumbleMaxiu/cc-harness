## 根目录 AGENTS.md 强制条款

每个根目录 AGENTS.md **必须**在 "关键约定" 或 "语言约束" 章节包含以下条款：

```markdown
## 语言约束

**必须使用中文输出**。所有回复、注释、提交信息均使用中文。
```

---

## 子目录 AGENTS.md 模板

子目录 AGENTS.md **不需要**语言约束条款（继承自根目录），专注于模块特定内容：

```markdown

> 本模块在项目中的职责

## 与根目录 AGENTS.md 的关系

本模块是 [父项目名] 的一部分，根目录 `AGENTS.md` 中的通用约定仍然适用。
```

## 质量检查清单

生成 AGENTS.md 后，自查：

- [ ] 行数在 50–200 之间
- [ ] 包含"快速开始"（1-3 条命令）
- [ ] 包含"关键约定"（3-8 条具体规则）
- [ ] 包含"文档地图"（指向 docs/ 文件）
- [ ] 根目录包含中文输出约束
- [ ] 没有复制粘贴整段代码或文档
- [ ] 没有过时信息（使用 `[最后更新: YYYY-MM-DD]`）
